FAQ

Frequently asked questions

How does pricing work?

Listing an event is free. Paid plans add boost credits, full analytics, and Discord delivery. Pay by card through Stripe's secure checkout, or with Robux via gamepass verification. See pricing.

How do reports work?

Connect your ER:LC server API key in Settings. When your event ends, Gatherly pulls your server data for the event window and compiles the full report. See a sample.

How long can events run?

Events run up to 90 minutes. When the end time passes, the listing leaves the discovery feed automatically and the report window closes.

Is my server data safe?

Your API key is encrypted at rest with AES-256 and only used to read statistics for your event window. Gatherly never issues commands to your server. Full details in the privacy policy.
